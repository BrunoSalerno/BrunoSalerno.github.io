Citylines.co

Citylines.co is a collaborative mapping platform where people can build the transport networks of their cities. The aim of the project is to host all the transit systems of the World, and their history, in one place. Currently there are more than 50 cities and +150 contributors. Features can be drawn in the built-in editor or imported directly from OpenStreetMap. Also, all the data is open and available to the public at citylines.co/data.

The stack is a Ruby/Sinatra server, a PostgreSQL/PostGIS database, and a React frontend. Map visualization is done with MapBox GL.

Sao Paulo 3D: Land Value

Sao Paulo 3D is both a reconstruction of the land value of each square of the Sao Paulo city in Brazil and a 3D visualization of this data, using Mapbox GL data-driven capability. The land value was calculated using the Properati Data datasets.

Greenness

Greenness is a project that I developed for Properati that had the scope of building an automatic greenness index using OpenStreetMap data. The closest to the coast or public parks, the "greennest" the area.
